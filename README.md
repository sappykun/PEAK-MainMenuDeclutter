# Main Menu Declutter

Removes stuff on the main menu.
Made against `1.52.a` - will not work if different buttons are added, and may break if existing ones are changed.

Thanks to letsalllovelain for helping me find the right GameObjects to remove.

## Configuration options

Default true:
- `BlockStoreButtons` removes the store buttons from the main menu.
- `BlockDeveloperButtons` removes the developer buttons from the main menu.
- `BlockDiscordButton` removes the Discord button from the main menu.

Default false:
- `BlockSpeechBubble` removes the developer comment speech bubble from the main menu.
- `BlockCreditsButton` removes the credits button from the main menu.
